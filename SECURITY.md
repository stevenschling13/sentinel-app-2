# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Sentinel Trading Platform, please report it responsibly.

**Do not open a public GitHub issue.**

Instead, email **stevenschlingman62@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

- You will receive an acknowledgment within **48 hours**
- We will work with you to understand and resolve the issue
- Once fixed, we will credit you in the release notes (unless you prefer anonymity)

## Scope

This policy applies to the Sentinel Trading Platform codebase and its deployed services.

## Bounty

There is no formal bug bounty program at this time.
